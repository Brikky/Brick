Typography

All typography is sized using rem units, where the default ratio is 1rem = 10px. Easily scale your text by changing the font-size of the HTML element.

The default font set is web-safe, and includes typical assets such as anchors, strong, emphasis, and underline.

Buttons

There are two default button stylings for Brick, <button> elements are plain, and .primary is brighter for increased visibility. Button classes can also be used on anchor and certain other tags to quickly add styling changes.
